# Mythral Graphics Overhaul — CraftPix Pixel Art Style

## Goal

Recreate all procedural pixel art assets in `client/src/art/` to match the quality, detail, and polish of CraftPix's top-down pixel art style (Grassland Tileset, Main Character's Home, and related free assets). The existing asset pipeline (generator → atlas → registry → renderer) stays intact — we only rewrite the **generator modules** and **palette** to produce higher-fidelity output.

## Reference Style Analysis (CraftPix)

From studying the Grassland Tileset and Main Character's Home images:

- **16px base tile grid** rendered at 32px (2x upscale) — crisp pixels, no anti-aliasing
- **Top-left light source** with consistent highlights/shadows across all assets
- **Rich texture detail**: grass has individual tufts, flowers, stones; water has wave highlights; dirt has cobble patterns
- **Seamless autotiling**: tiles connect smoothly with neighbor-rule transitions (grass→dirt, grass→water edges)
- **Detailed buildings**: stone foundations, wooden walls with plank lines, tiled roofs with individual shingles, windows with glass panes, arched doorways
- **Lush vegetation**: layered tree canopies (3-4 shade layers), berry bushes, mushroom clusters, vine-covered ruins
- **Consistent palette**: warm earthy tones for grassland, cohesive color ramps per material
- **Shadow depth**: objects cast subtle shadows, depth via layering (darker at base, lighter at top)

## Implementation Plan

### Phase 1: Palette & Foundation (`palette.js`, `pixelbuf.js`)

**File: `client/src/art/palette.js`**
- Expand palette ramps to 6-7 stops (deep shadow → shadow → base → mid → light → highlight → specular)
- Add CraftPix-matched color families:
  - Grass: warmer greens with yellow undertones (#5a9e3f → #8fd66a range)
  - Water: deeper blues with foam highlights
  - Stone: cooler grays with warm moss tones
  - Wood: richer browns with grain variation
  - Roof tiles: terracotta reds with individual shingle colors
- Add material-specific sub-palettes (grass tuft colors, flower varieties, stone textures)
- Keep backward compatibility with existing TILE_INFO sprite keys

**File: `client/src/art/pixelbuf.js`**
- Add new drawing primitives needed for detailed pixel art:
  - `ditherRect()` — ordered dithering for texture gradients
  - `noiseRect()` — per-pixel noise fill with color ramp mixing
  - `bevelRect()` — 3D beveled rectangle with highlight/shadow edges
  - `roundRect()` — rounded rectangle for organic shapes
  - `gradientFill()` — vertical/horizontal gradient with noise

### Phase 2: Tile Generator Rewrite (`gen-tiles.js`)

Rewrite every tile painter to produce CraftPix-quality output:

**Grass tiles (4 variations each):**
- Base grass with individual blade tufts (3-4 pixel tall grass blades)
- Scattered flowers (5-6 colors: pink, red, yellow, blue, purple, white)
- Small stones and pebbles with highlight dots
- Clover patches and moss details
- Smooth noise texture with warm green variation

**Water tiles (3 variations):**
- Wave highlight lines (2-3 pixel horizontal streaks)
- Foam near shorelines
- Subtle ripple patterns
- Deep water gradient with darker center

**Path/dirt tiles (3 variations):**
- Cobblestone pattern with individual stones
- Dirt clumps and grass edge blending
- Worn path center with lighter highlights
- Edge transition tiles for grass↔path

**Stone/wood floor tiles:**
- Individual plank lines with grain detail
- Stone brick pattern with mortar lines
- Wood knot details
- Wear and age texture

**Wall/door tiles:**
- Brick pattern with offset rows
- Stone texture with moss in crevices
- Door with wood grain, handle, and frame
- Window cutouts with glass panes

**Forest/mountain tiles:**
- Tree canopy with 3-layer depth (dark base → mid → light top)
- Individual leaf clusters
- Mountain with snow cap and rock texture
- Dense forest with overlapping canopies

**New: Autotile edge variants**
- Generate corner/edge transition tiles for each terrain type
- Grass→water, grass→dirt, grass→stone transitions
- Smooth blending at terrain boundaries

## Rollout Strategy

All biomes and all asset categories will be tackled simultaneously in a single coordinated effort. The generators will be rewritten to support all 9 biomes (Lumina/grassland, Emberfall/volcanic, Frostpeak/snow, Mistwood/magic forest, Sunscar/desert, Tidehaven/coastal, Shadowfen/swamp, Skyreach/cloud, Voidheart/void) with biome-specific palette shifts and tile sets.

## Risk Mitigation

- **Backward compatibility**: All existing sprite keys (`tile.grass.0`, `char.warrior.walk.down.0`, etc.) are preserved. The generators produce the same keys with better art.
- **Performance**: New generators may produce larger atlases. Monitor atlas sizes and split if needed.
- **Biome palette shifts**: Each biome reuses shared material ramps with hue/value shifts via `BIOME_TINT` in `palette.js` — no need to redesign per-biome from scratch.
